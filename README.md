# Flight Booking App ✈️

This is a full-stack flight booking system built with:

- **Backend**: Node.js, Express, MongoDB
- **Frontend**: React

## Features
- User login/signup
- Flight search and filters
- Seat selection
- Booking history
- Admin panel
- Wallet and refund logic

## How to run
1. Clone the repo
2. Add your `.env` file in `Backend/`
3. Run backend: `node server.js`
4. Run frontend: `npm start` inside `Frontend/`
